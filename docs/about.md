# About RedShelf

RedShelf is a static academic PDF library designed for researchers, postgraduate students, evidence teams and knowledge-management projects. It combines the clarity of modern documentation with the browsing patterns of a digital research library.

## What RedShelf provides

- Fast, client-side full-text site search
- Topic, category and tag-based browsing
- Consistent scholarly metadata and citations
- In-browser PDF previews and direct downloads
- Responsive light and dark interfaces
- A maintainable Markdown-first publishing workflow
- Automated deployment to GitHub Pages

## Static by design

RedShelf has no backend and no database. Documents, metadata and navigation remain version-controlled with the project. This makes the library easy to audit, archive, fork and host.

## Repository

The project source is available at [saprimad/redshelf](https://github.com/saprimad/redshelf). Contributions can be proposed through issues and pull requests.

## Content responsibility

Only upload PDFs that you are permitted to store and redistribute. Review copyright, licensing, privacy and research-ethics requirements before publishing any document.
